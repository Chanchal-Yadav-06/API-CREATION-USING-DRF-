# Django REST API 🔗

A REST API built with Django REST Framework (DRF)
with full CRUD operations.

## Features
- Full CRUD via REST APIs
- Token based structure
- Tested with Postman

## Tech Stack
- Python
- Django
- Django REST Framework
- MySQL

## How to Run
```bash
git clone https://github.com/Chanchal-Yadav-06/API-CREATION-USING-DRF-
cd API-CREATION-USING-DRF-
pip install -r requirements.txt
python manage.py runserver
```
